# GPI Website

This repository contains the source code for the **GPI Website**, built using **React**, **TypeScript**, and **Vite**.
The project uses modern tooling, fast development workflows, and a clean structure to help new contributors get started quickly.

---

## 🚀 Features

* ⚡ **Vite** for lightning-fast development and bundling
* ⚛️ **React + TypeScript** for type-safe UI development
* 🎨 Easily customizable component structure
* 🧩 Modular and scalable architecture for long-term growth

---

## 📦 Getting Started

### 1. **Prerequisites**

Make sure you have the following installed:

* **Node.js** (v18 or newer recommended)
* **npm**, **pnpm**, or **yarn**

### 2. **Install Dependencies**

Using npm:

```bash
npm install
```

or with pnpm:

```bash
pnpm install
```

### 3. **Run the Development Server**

```bash
npm run dev
```

or with pnpm:

```bash
pnpm run dev
```

The project will be available at:

```
http://localhost:5173
```

### 4. **Build for Production**

```bash
npm run build
```

### 5. **Preview Production Build**

```bash
npm run preview
```
